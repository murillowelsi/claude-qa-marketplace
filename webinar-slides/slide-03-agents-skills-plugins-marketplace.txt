================================================================================
  SLIDE 3: THE NEW BUILDING BLOCKS — Agents, Skills, Plugins, Marketplaces

================================================================================

  MARKETPLACE
  -----------
  A curated registry where teams publish and discover Plugins.
  Instead of every team reinventing QA prompts from scratch,
  you browse, install, and start using proven Skills immediately.

  The Marketplace hosts the QA Toolkit and more.

  PLUGIN
  ------
  A collection of related Skills bundled together under one namespace.
  A Plugin is what you install to give your agent a full capability domain.

  Example: the "QA Toolkit" plugin gives your agent all five QA skills
  in one install — no configuration needed.

  SKILL
  -----
  A reusable, focused capability you give to an Agent.
  A Skill is a prompt + instructions + tool permissions, packaged so
  the agent knows exactly how to complete a specific QA task.

  Examples: analyze a story, generate test cases, assess risk, report a bug.
  You invoke a Skill with a slash command: /analyze-story, /generate-test-cases

  HOW THEY CONNECT:
  -----------------
  Marketplace  -->  Plugin  -->  Skills  -->  Your QA Workflow
